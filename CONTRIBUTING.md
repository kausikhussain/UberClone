# Contributing to UberClone

Thank you for your interest in contributing to the UberClone project!

## How to Contribute

1. Fork the repository.
2. Create a new branch for your feature or bugfix.
3. Make your changes and ensure they follow the project style.
4. Submit a Pull Request with a clear description of your changes.

## Code of Conduct

Please be respectful and professional in all communications.

## Development Setup

1. Clone the repo.
2. Run `npm install` in both `client` and `server` directories.
3. Set up your `.env` files based on the `.env.example` templates.
4. Run `npm run dev` to start the development servers.
